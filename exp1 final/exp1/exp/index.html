Instructions

  1. Select the number of processes (P) and number of resources (R) and click the SUBMIT button to initialize the data structures, including the Allocation Matrix, Available Matrix (vector), and Maximum Matrix.
  2. Enter the calculated values into the Need Matrix using the following formula:
    Need matrix = Maximum matrix - Allocation matrix
  3. Click on the CHECK button to verify the need matrix.
  4. Click on the STARTbutton to initialize the Work and Finish Vectors.
  5. Click on the NEXT button to:
    • Check if the need of process Pi can be satisfied with the current Work vector using:
      Needi <= Work
  6. Repeat Step 5 until you obtain either a safe or unsafe sequence based on the completion status of the processes.

    7. NOTE 1: The highlighted row in the Allocation Matrix indicates that the process Pi has been included in the safe sequence.
    NOTE 2: Read the alerts carefully.
